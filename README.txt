IMAGE ANONYMIZATION PIPELINE

The script “image_anonymization_code.m” shows an example of image anonymization pipeline.
Author: Sara Mandelli

BEFORE RUNNING THE CODE:

1. DOWNLOAD THE PACKAGES OF BM3D AND NOISE EXTRACTION FROM THE WEBSITES:

“http://www.cs.tut.fi/~foi/GCF-BM3D/#ref_software”
Copy the content of all the package in the folder “BM3D”

“http://dde.binghamton.edu/download/camera_fingerprint/“
Copy the content of all the package in the folder “CameraFingerprint”

2. RUN THE FUNCTION “compile.m” IN THE FOLDER “CameraFingerprint/Filter”

 
Reference: S. Mandelli, L. bondi, S. Lameri, V. Lipari, P. Bestagini, S. Tubaro,
"Inpainting-Based Camera Anonymization"
IEEE International Conference on Image Processing (ICIP), 2017.
